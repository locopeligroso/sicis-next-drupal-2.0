# Mega-menu Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Implementare un mega-menu a colonne orizzontali nella navbar di Sicis, con apertura hover su desktop e tap su mobile, usando i dati reali del menu Drupal a 3 livelli.

**Architecture:** `transformMenuToNavItems` viene esteso per mappare ricorsivamente tutti i livelli di children. Un nuovo componente `MegaMenu.tsx` gestisce il pannello a colonne. `Header.tsx` viene refactored per gestire la logica hover/click adattiva e renderizzare `MegaMenu` condizionalmente.

**Tech Stack:** Next.js 14 App Router, TypeScript strict, React hooks (`useState`, `useEffect`, `useRef`, `useCallback`), `next/navigation` (`usePathname`), CSS inline con unità rem/em/%

**Spec:** `docs/superpowers/specs/2026-03-18-mega-menu-design.md`

- [ ] **Step 2: Aggiungere funzione helper ricorsiva**

Aggiungi questa funzione privata PRIMA di `transformMenuToNavItems` in `src/lib/fetch-menu.ts`:

```typescript
/**
 * Recursively transforms a single menu item and all its descendants.
 * Normalises URLs at every level so locale prefix is always correct.
 */
function transformItemRecursive(item: MenuItem, locale: string): MenuItem {
  return {
    ...item,
    url: normalizeUrl(item.url, locale),
    children: item.children.map((child) => transformItemRecursive(child, locale)),
  };
}
```

- [ ] **Step 3: Aggiornare `transformMenuToNavItems` per usare l'helper**

Sostituisci il corpo di `transformMenuToNavItems`:

```typescript
export function transformMenuToNavItems(
  menu: Menu | null,
  locale: string
): MenuItem[] {
  if (!menu?.items) return [];
  return menu.items.map((item) => transformItemRecursive(item, locale));
}
```

**Struttura dati in input:**

## Chunk 2: Header.tsx refactor

### Task 3: Refactor `Header.tsx` — logica hover/click adattiva + render MegaMenu

**Files:**
- Modify: `src/components/layout/Header.tsx`

**Contesto:**
`Header.tsx` è un client component (~160 righe). Attualmente ha:
- State `openIndex: number | null`
- Logica hover con `onMouseEnter`/`onMouseLeave` e debounce 120ms
- Render dropdown semplice per `item.children`

Va refactored per:
1. Rilevare `isTouch` al mount via `matchMedia('(pointer: fine)')`
2. Su desktop (non touch): hover apre/chiude con debounce 150ms
3. Su mobile (touch): click toggling + click fuori chiude
4. Renderizzare `<MegaMenu>` invece del dropdown inline
5. Evidenziare la voce attiva via `usePathname()`

**Struttura menu a 3 livelli:**
- Livello 0: EXPLORE, FILTER & FIND, PROJECTS, Home (voci navbar)
- Livello 1: Mosaico, Vetrite, Furniture... (children di EXPLORE — colonne del mega-menu)
- Livello 2: Antigua, Gem Glass... (children delle colonne — link nel mega-menu)

Il mega-menu si apre quando `openIndex` corrisponde a una voce che ha `children.length > 0`.

Leggi `src/components/layout/Header.tsx` per capire la struttura esistente prima di modificare.

- [ ] **Step 3: Aggiornare lo state e i ref**

**Nota sui tipi**: `initialMenu` è già `MenuItem[]` (già trasformato, passato da layout.tsx). NON va ri-trasformato. `Menu` ha tipo `{ id: string; items: MenuItem[] }` — usato solo per il fetch client-side.

Sostituisci la sezione state/ref del componente con:

```typescript
const [menuItems, setMenuItems] = useState<MenuItem[]>(initialMenu ?? []);

const [openIndex, setOpenIndex] = useState<number | null>(null);
const [isTouch, setIsTouch] = useState(false);
const closeTimer = useRef<ReturnType<typeof setTimeout> | null>(null);
const pathname = usePathname();
```

- [ ] **Step 4: Aggiungere useEffect per fetch menu e rilevamento touch**

Aggiungi dopo lo state:

```typescript
// Fetch menu se non fornito come prop
useEffect(() => {
  if (initialMenu) return;
  fetchMenu('main', locale).then((menu) => {
    if (menu) setMenuItems(transformMenuToNavItems(menu, locale));
  });
}, [locale, initialMenu]);

// Rileva touch device una sola volta al mount
useEffect(() => {
  setIsTouch(!window.matchMedia('(pointer: fine)').matches);
}, []);

// Chiudi mega-menu su click fuori (solo touch).
// NOTA: il listener viene aggiunto DOPO che React ha processato il batch di state update
// che ha aperto il menu, quindi non intercetta il tap di apertura. Questo è safe.
// stopPropagation in handleNavClick previene comunque la chiusura immediata.
useEffect(() => {
  if (!isTouch || openIndex === null) return;
  const handleClickOutside = () => setOpenIndex(null);
  document.addEventListener('click', handleClickOutside);
  return () => document.removeEventListener('click', handleClickOutside);
}, [isTouch, openIndex]);
```

- [ ] **Step 5: Aggiungere handler hover/click**

Aggiungi le funzioni handler:

```typescript
const clearCloseTimer = useCallback(() => {
  if (closeTimer.current) {
    clearTimeout(closeTimer.current);
    closeTimer.current = null;
  }
}, []);

const startCloseTimer = useCallback(() => {
  clearCloseTimer();
  closeTimer.current = setTimeout(() => setOpenIndex(null), 150);
}, [clearCloseTimer]);

const handleNavMouseEnter = useCallback(
  (index: number) => {
    if (isTouch) return;
    clearCloseTimer();
    setOpenIndex(index);
  },
  [isTouch, clearCloseTimer]
);

const handleNavMouseLeave = useCallback(() => {
  if (isTouch) return;
  startCloseTimer();
}, [isTouch, startCloseTimer]);

const handleNavClick = useCallback(
  (e: React.MouseEvent, index: number, hasChildren: boolean) => {
    if (!isTouch || !hasChildren) return;
    e.preventDefault();
    e.stopPropagation(); // evita che il click fuori chiuda subito
    setOpenIndex((prev) => (prev === index ? null : index));
  },
  [isTouch]
);

const handleMenuClose = useCallback(() => setOpenIndex(null), []);
```

- [ ] **Step 6: Aggiungere helper per voce attiva**

```typescript
const isActiveItem = useCallback(
  (item: MenuItem) => {
    if (!item.url || item.url === `/${locale}`) return false;
    return pathname.startsWith(item.url);
  },
  [pathname, locale]
);
```

```typescript
return (
  <header
    style={{
      position: 'sticky',
      top: 0,
      zIndex: 100,
      background: '#fff',
      borderBottom: '0.0625rem solid #e0e0e0',
    }}
  >
    <nav
      style={{

        margin: '0 auto',
        padding: '0 2rem',
        display: 'flex',
        alignItems: 'center',
        gap: '2rem',
        height: '3.5rem',

      {/* Logo */}
      <Link
        href={`/${locale}`}
        style={{ fontWeight: 800, fontSize: '1.1rem', letterSpacing: '0.1em', color: '#111', textDecoration: 'none' }}
      >
        SICIS
      </Link>

      {/* Voci navbar */}
      <ul
        role="menubar"
        style={{ display: 'flex', gap: '1.5rem', listStyle: 'none', margin: 0, padding: 0 }}
      >
        {menuItems.map((item, i) => {
          const hasChildren = item.children.length > 0;
          const isOpen = openIndex === i;
          const isActive = isActiveItem(item);

          return (
            <li key={item.id} style={{ position: 'relative' }}>
              <Link
                href={item.url || `/${locale}`}
                role="menuitem"
                aria-haspopup={hasChildren ? 'menu' : undefined}
                aria-expanded={hasChildren ? isOpen : undefined}
                onMouseEnter={() => handleNavMouseEnter(i)}
                onMouseLeave={handleNavMouseLeave}
                onClick={(e) => handleNavClick(e, i, hasChildren)}
                style={{
                  fontSize: '0.75rem',
                  fontWeight: 600,
                  letterSpacing: '0.1em',
                  textTransform: 'uppercase',
                  color: isActive ? '#000' : '#444',
                  textDecoration: 'none',
                  borderBottom: isActive ? '0.125rem solid #000' : '0.125rem solid transparent',
                  paddingBottom: '0.2rem',
                  display: 'block',
                }}
              >
                {item.title}
              </Link>

              {/* Mega-menu panel */}
              {hasChildren && isOpen && (
                <MegaMenu
                  items={item.children}
                  onClose={handleMenuClose}
                  onMouseEnter={clearCloseTimer}
                  onMouseLeave={startCloseTimer}
                  locale={locale}
                />
              )}
            </li>
          );
        })}
      </ul>
    </nav>
  </header>
);
```
