REGOLE TRASVERSALI

Comportamento filtri & interazioni

P0 vince su P1/P2: Colore/Collezione (radio, uno alla volta). Se seleziono un P0 che rende un P1 incompatibile (count=0), il P1 viene automaticamente rimosso. Feedback: chip scompare con transizione.
Filtri sempre aperti: Nessun accordion/collapsible. Tutte le sezioni sempre visibili.
No stucco: Rimosso dai filtri Mosaico e Pixall.
Swatches colore nel pannello: Senza label testuale, solo cerchi. aria-label per accessibilità.
Container: Hub + Master = max-w-main (1280px). Listing con filtro = max-w-listing (1536px).
Pannello glass: Sticky (V1: top semplice sotto navbar ~80px). Bi-directional sticky come enhancement futuro.
Mobile: Pannello → Sheet laterale con scroll interno. FAB fixed bottom center. Context bar/airy header adattati.
Breadcrumb: "Filter&Find › Categoria ▾ › Sottocategoria". Dropdown per cambiare categoria.
"Cambia" popover: Collezioni = lista verticale con thumbnail. Colori = griglia swatches senza nomi.
Condividi: 🔗 copia URL corrente (con filtri) negli appunti + toast conferma.
